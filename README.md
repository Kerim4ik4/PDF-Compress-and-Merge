# PDF-Compress-and-Merge
Free efficient PDF merge and compress tool. Merges and compresses PDF files better than any online services. 

Unpack the archive to the Desktop.
Run Start.bat file.
GUI will appear. Enjoy!)
Uses official binaries of: (see; LIBS/ directory) 
qpdf
pdftk
ghostscript

As GUI uses powershell script (see; LIBS\ directory) 

Tested on Windows 10.
